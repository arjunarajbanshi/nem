> ## NODE JS

- Node js is a javascript runtime built on google's open-source v8 javascript engine.

- Javascript outside of browser.

- Single threaded, based on event driven, non-blocking I/O model.

### Running Javascript/Nodejs inside terminal

- Open terminal

- Type `node`, you will enter node interactive mode

- Now you can type javascript outside of browser using nodejs

- If you hit `TAB`inside node, you will see all global variables of nodejs.

- More...

  ```bash
  node # To enter node interactive mode
  2+2 # Enter
  4
  _-4 # using underscore _ we can access previous result
  0 # 4-4
  String. # Hit TAB and you will see all string method available
  ```

- To exit : type `.exit`
